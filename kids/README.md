# InsightFlow Kids

Daily personalized learning packets for Dahlia and Xander.

## Components

- `skill/` - OpenClaw skill for daily generation
- `content/` - Activity databases (puzzles, riddles, etc.)
- `generators/` - PDF creation modules
- `profiles/` - Child profiles and feedback tracking
